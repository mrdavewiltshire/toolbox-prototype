Trade Partner ToolBox: Typography Best Practice

Same components, same data, same brand. The only change is the type treatment, and every size now sits on the Microsoft Fluent 2 type ramp. The rule of thumb: labels whisper, values speak, and only section headers get colour.

Rule 1
Labels are Caption 1: small, regular weight, muted grey. They identify the data, they never compete with it.
Rule 2
Values are Subtitle 2: larger, semibold, near-black. The answer should always dominate the question.
Rule 3
One semibold element per component. Fluent uses semibold (600) for emphasis, never heavy bold.
Rule 4
Colour marks navigation and section headers only. Data stays neutral so the eye maps colour to structure.
